## 🚀 Proyectos Destacados

### 🐦 BirdScope — Clasificación de Aves con IA

> Aplicación móvil híbrida con inferencia de modelos de Deep Learning directamente en el dispositivo.

**Stack:** `Ionic` `Angular` `TensorFlow.js` `Firebase` `Leaflet` `OpenStreetMap`

| Funcionalidad | Descripción |
|---|---|
| 🤖 IA On-Device | Clasificación en tiempo real sin servidor |
| 📍 Geolocalización | Registro con coordenadas GPS precisas |
| 🗺️ Mapas interactivos | Visualización de avistamientos con OpenStreetMap |
| 📊 Reportes | Estadísticas y análisis de datos |
| 🔥 Firebase | Sincronización y almacenamiento en la nube |

---

### 👷 TalentGo — Marketplace de Trabajo Independiente

> Plataforma móvil que conecta trabajadores independientes con contratistas, con sistema de reputación integrado.

**Stack:** `Flutter` `Firebase` `Firestore` `Cloud Storage`

| Funcionalidad | Descripción |
|---|---|
| 🧑‍💼 Perfiles | Gestión de usuarios y portafolio laboral |
| 📋 Solicitudes | Publicación y aplicación a ofertas de trabajo |
| ⭐ Calificaciones | Sistema de reputación bidireccional |
| 💬 WhatsApp | Integración directa para contacto |
| 🔒 Seguridad | Reglas de Firestore con autenticación |
