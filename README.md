# 🚀 Actualmente trabajando en

## 🐦 BirdScope

Aplicación móvil híbrida desarrollada con **Ionic + Angular + TensorFlow.js** para la clasificación inteligente de aves mediante IA.

### Tecnologías utilizadas

- Ionic
- Angular
- TensorFlow.js
- Firebase
- Leaflet
- OpenStreetMap
- Deep Learning

### Funcionalidades

- 📷 Clasificación de aves en tiempo real
- 🧠 Inferencia IA On-Device
- 📍 Geolocalización GPS
- 🗺️ Mapas interactivos
- ☁️ Firebase en tiempo real
- 📊 Reportes y estadísticas
- 📱 Compatibilidad Android

## 👷 TalentGo

Aplicación móvil desarrollada en **Flutter + Firebase** para conectar trabajadores independientes con contratistas.

### Funcionalidades

- 👷 Registro de trabajadores
- 🧑‍💼 Registro de contratistas
- 📄 Solicitudes laborales
- ⭐ Sistema de reseñas y calificaciones
- 🔒 Seguridad en Firestore
- 📷 Galería de trabajos
- 📱 WhatsApp Integration
- 🔥 Backend en Firebase
